# Installation Scheduling App

## Trigger

Option 1 Reccomended
Item delivery (in BigCommerce) triggers email sent to customer to book installation scheduling.
Link to this installation scheduling web app book installation. Details including Name, email and order number passed to the form.

Option 2 Fallback
User receives link to book installation but must acknoledge they have received the item.
User must book installation
1st: Acknowledge that delivery has been received.
2nd: Book installation using email and order number passed to the form from email.
